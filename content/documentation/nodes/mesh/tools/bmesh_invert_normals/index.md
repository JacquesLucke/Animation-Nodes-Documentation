## Description

This node invert the normals of vertices and polygons of the input bmesh.

## Inputs

- **Bmesh** - An Input Bmesh.
- **Flip Multires** - This option if enabled will interpolate the loops multires data.

## Outputs

- **Bmesh** - The resulted Bmesh.
